**Introduction:**
This was my senior design project, where we attached an LED matrix, a microcontroller, and a battery pack onto a hat to create a wearable piece of tech that can display color and text across the hat. The LED matrix is powered by a 18650 battery cell and is controlled by an Arduino microcontroller. The hat is able to light up multiple colors, enter rainbow mode, display text, and dance to music. There is an app that can control the lights on the hat.

**Hardware:** \
Computing: 
* Ardunio microcontroller
* 18650 battery cell
* Flexible LED matrix
* 18650 battery management module
